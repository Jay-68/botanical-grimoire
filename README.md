# Botanical Grimoire

A mystical plant encyclopedia built with React and Vite. Explore the hidden lore, uses, and properties of the botanical world.

## Features

- Browse and discover a curated collection of plants and herbs
- Detailed entries on botanical properties, folklore, and uses
- Clean, responsive UI styled with Tailwind CSS
- Smooth navigation powered by React Router
- Iconography via Lucide React

## Tech Stack

- **React 18** — UI framework
- **Vite** — lightning-fast build tool
- **Tailwind CSS** — utility-first styling
- **React Router v6** — client-side routing
- **Lucide React** — icon library

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
git clone https://github.com/Jay-68/botanical-grimoire.git
cd botanical-grimoire
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
